Tables

Table Tags

Tag Name Description
<table> Table Element that represents data with more than one dimension, in the form of a table.
<thead> Table Head (rows) Element that represents the block of rows that consist of the column labels (headers) for the parent table element.
<th> Table Head (cell) Element that represents a header cell in a table.
<tbody> Table Body Element that represents a block of rows that consist of a body of data for the parent table element.
<tr> Table Row Element that represents a row of cells in a table.
<td> Table Data Element that represents a data cell in a table.
<tfoot> Table Foot Element that represents the block of rows that consist of the column summaries (footers) for the parent table element.

Table Attributes

Attribute Name Description
<colspan> Column Span Defines how many columns a td or th element should span.
<rowspan> Row Span Defines how many rows a td or th element should span.

Input

Main Input Types

Type Description Example
button A push button with no default behavior displaying the value of the value attribute, empty by default.
checkbox A check box allowing single values to be selected/deselected.
color A control for specifying a color; opening a color picker when active in supporting browsers.
date A control for entering a date (year, month and day).
file A control that lets the user select a file.
number A control for entering a number. Displays a spinner and adds default validation when supported.
password A single-line text field whose value is obscured. Will alert user if site is not secure.
radio A radio button, allowing a single value to be selected out of multiple choices with the same name value.
range A control for entering a number whose exact value is not important.
submit A button that submits the form. Displays 'Submit' by default
text The default value. A single-line text field.
